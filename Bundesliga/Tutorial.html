Passo a Passo

  1. Clique no botão "Gerar Partidas" para criar os jogos.
  2. Acompanhe os resultados dos jogos e veja a tabela de classificação.
  3. Use o botão "Mostrar/Ocultar Jogos" para exibir ou esconder os jogos gerados.
  4. O passo a passo pode ser acessado a qualquer momento clicando no botão "Como Jogar".
